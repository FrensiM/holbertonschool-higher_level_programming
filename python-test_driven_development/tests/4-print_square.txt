Module 5-text_indentation(text)

def text_indentation(text) Prints the given text while being formated

	>>> text_indentation = __import__('5-text_indentation').text_indentation

Text must be string

	>>> text_indentation(2)
	Traceback (most recent call last):
	TypeError: text must be a string

Function should print formated text

	>>> text_indentation("Ckemi Frensi. Me thua dot sa eshte ora? Ora eshte: 10")
	Ckemi Frensi.
	<BLANKLINE>
	Me thua dot sa eshte ora?
	<BLANKLINE>
	Ora eshte:
	<BLANKLINE>
	10

Even when there are lots of spaces

	>>> text_indentation("Ckemi Frensi.    Me thua dot sa eshte ora?   11")
	Ckemi Frensi.
	<BLANKLINE>
	Me thua dot sa eshte ora?
	<BLANKLINE>
	11
